Компиляция java кода из командной строки

1. В папке src (в ней будут храниться исходные файлы) 
создаём файл TestClass (в пакете src), содержащий 
класс с таким же названием и методом main. 
Сохраняем с расширением .java 

2. Создаём отдельную папку /bin для скомпилированных классов.

3. Чтобы скомпилировать класс, 
находящийся в пакете src, необходимо вызвать командную строку
из корнейвой папки (Homework_1) и выполнить команду
javac -d bin ./src/TestClass.java

-d - указывает на папку, в которую необходимо поместить 
скомпилированные файлы (в данном случае bin)
./src/ - путь к исходным файлам. * - скомпилировать все файлы в папке

Скомпилированный класс TestClass.class появится в папке bin/src

4. Для запуска используется команда
java -classpath ./bin src.TestClass

-classpath - указывает путь к скомпилированным классам
./bin - папка, в которой лежат скомпилированные классы
src.TestClass  - название пакета и название класса


Создание jar архива

1. В главной папке (Homework_1) создаём файл manifest.mf, который 
будет описывать jar-файл.

2. Указываем в данном файле версию (manifest-version: 1.0) путь к классу с методом 
main (src.TestClass) и путь к уже скомпилированным файлам 
(папка bin TestClass.class)

3. Создаём jar-архив командой 
jar -cmf manifest.mf TestClass.jar -C bin .

c - создать новый архив
m - добавить существующий файл манифеста 
(после флагов указывается путь к данному файлу) 
f - указывается имя архива (TestClass.jar)
- C - указывает путь к скомпилированным файлам (папка bin)
. - путь к jar-архиву (в данном случе - корневой каталог)

4. Запускаем jar-архив командой
java -jar TestClass.jar

-jar - флак указывает, что запускается именно jar-файл
TestClass.jar - путь к архиву (леэит в корневом каталоге, поэтому 
только название)
